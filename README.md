
# APT1030-A: Fundamentals of Programming Languages
**Student Capstone Portfolio**

This repository contains a structured progression through the core concepts of Programming
, applied to real-world Kenyan digital systems (eCitizen, NHIF, SACCOs, and Ride-Hailing).

---

## Repository Structure

### [Level 1: Syntax & Semantics](./Level 1 - Login Validation System/)
* **Context:** eCitizen Login Validation.
* **Concepts:** Compilation vs. Interpretation, Syntax/Logic errors.

### [Level 2: Data Types](./SACCO member financial record/)
* **Context:** SACCO Member Financial Records.
* **Concepts:** Static vs. Dynamic typing, Type Coercion.

### [Level 3: Control Flow](./Level 3 - Drought advisory logic/)
* **Context:** Machakos County Drought Advisory.
* **Concepts:** Nested Logic, Switch Statements.

### [Level 4: Abstraction](./Level 4 - NHIF Claims Processing/)
* **Context:** NHIF Claims Processing.
* **Concepts:** OOP, Encapsulation, Classes.

### [Level 5: Paradigms](./Level - 5 Ride Hailing Pricing Engine/)
* **Context:** Nairobi Ride-Hailing Pricing Engine.
* **Concepts:** Procedural vs. OOP vs. Functional (Lambdas).

### [Level 6: Scope & Memory](./Level - 6 patient record access control/)
* **Context:** Patient Record Access Control.
* **Concepts:** Exception Handling, Lexical Scope.

### [Level 7: Capstone Implementation](./Level - 7 Mini DSL for Mobile money /)
* **Context:** Mini DSL for Mobile Money Rules.
* **Concepts:** Lexing, Parsing, and Interpreter Logic.
